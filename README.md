Idle Alert is an application that notifies you whenever your character becomes idle.

How to run Idle Alert:
1) Download ZIP.
2) Extract the folder.
3) Open the Idle Alert.exe properties → Compatibility → [✓] Run this program as an administrator → Apply → OK.
4) Do not move any files or folders out of the Idle Alert folder. It is highly recommended that you create a shortcut to Idle Alert.exe.
6) Run it—and you're done!
